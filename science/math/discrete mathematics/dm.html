Theorem 1

A set with \( n \) elements has \(2^n\) subsets.

Theorem 2

The number of strings of length \( n \) composed of k given elements is \( k^n \).

Theorem 3

Suppose that we want to form strings of length \( n \) so that we can use any of a given set of \( k_1 \) symbols as the first element of the string, any of a given set of \( k_2 \) symbols as the second element of the string, etc., any of a given set of \( k_n \) symbols as the last element of the string. Then the total number of strings we can form is \( k_1 \cdot k_2 \cdot \dotso \cdot k_n \).

Theorem 4. Permutations

The number of permutations of \( n \) objects in \( n! \).

Theorem 5. [Stirling’s Formula]

\( n! \sim \left(\frac{n}{e}\right)^n \sqrt{2 \pi n} \)

Theorem 6

A set of \( n \) lines in general position in the plane divides the plane into \( 1 + n(n+1)/2 \) regions.

Theorem 7

The number of ordered \(k\)-subsets of an \(n\)-set is \( n(n−1). . . (n−k +1) = \frac{n!}{(n-k)!} \)

Theorem 8

The number of \(k\)-subsets of an \(n\)-set is \( \frac{n(n-1)...(n-k+1)}{k!} = \frac{n!}{k!(n-k)!} = \binom{n}{k} \)

Theorem 9. Binomial Theorem

The coefficient of \( x^k y^{n−k} \) in the expansion of (x+y)^n is \binom{n}{k}.
